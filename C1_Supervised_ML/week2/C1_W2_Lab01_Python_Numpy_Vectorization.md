# C1_W2_Lab01 - Python, Numpy and Vectorization

## 核心知识点

### 1. Numpy 数组基础
- `np.zeros(n)` 创建全0数组，`np.arange(n)` 创建 0 到 n-1 的数组
- 一维数组的 shape 是 `(n,)`，如 `np.zeros(5).shape` 是 `(5,)` 而不是 `(1,5)`

### 2. Shape 属性
- shape 是数组的属性（不是函数），返回一个元组描述每个维度的大小
- `(5,)` — 一维，5个元素
- `(2, 3)` — 二维，2行3列
- `()` — 0维，标量（scalar）
- 从数组取单个元素（如 `a[2]`）得到标量，shape 为 `()`

### 3. 一维数组 vs 二维矩阵的 shape 区分
- `(4,)` 是一维向量
- `(1, 4)` 是二维矩阵（1行4列）
- 这两个是不同的东西，不要混淆

### 4. 索引和切片
- 索引取单个元素：`a[2]` 返回标量
- 切片是**左闭右开**：`a[2:5]` 取索引 2, 3, 4
- 完整格式：`a[start:stop:step]`，步长为1时可省略
- 二维数组：`a[行, 列]`，如 `a[2, 1:4]` 取第2行的列索引 1, 2, 3

### 5. Reshape
- `.reshape(-1, 2)` 把数组重新排列成矩阵，-1 表示让 numpy 自动计算该维度的大小
- 例：`np.arange(6).reshape(-1, 2)` 得到 shape `(3, 2)` 的矩阵

### 6. 点积（Dot Product）
- `np.dot(a, b)` 计算两个向量的点积，结果是标量，shape 为 `()`
- 例：`np.dot([1,2,3], [4,5,6])` = 1*4 + 2*5 + 3*6 = 32

### 7. 向量化（Vectorization）
- 核心优势是**速度快**，不是功能上的区别
- np.dot 等向量化操作底层用 C 语言实现，能利用 CPU 的并行指令（SIMD）
- 比 Python for 循环快几百倍，数据量越大优势越明显

## 出错的地方

1. **`np.zeros(5)` 的 shape**：误答 `(1,)`，正确是 `(5,)` — 一维数组有5个元素
2. **一维数组 shape 写法**：`np.array([1,2,3,4]).shape` 误答 `(1,4)`，正确是 `(4,)` — 不要把一维数组和二维矩阵搞混
3. **切片左闭右开**：`a[2, 1:4]` 误算包含了索引4的值，实际只到索引3
4. **向量化的意义**：误理解为"整合多个特征"，实际核心是计算速度的提升

## 学生提问

- **shape 是什么**：是数组属性，返回元组描述维度大小，标量的 shape 是空元组 `()`
- **reshape 中 -1 的含义**：让 numpy 自动计算该维度的大小
- **切片是否左闭右开**：是的，`start:stop:step` 中 stop 不包含
