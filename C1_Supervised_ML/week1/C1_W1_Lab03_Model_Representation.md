## 核心知识点

### 1. 线性回归模型
$$f_{w,b}(x) = wx + b$$
- `w`（weight，权重）：控制直线的**斜率**，决定 y 随 x 变化的速率
- `b`（bias，偏置）：控制直线的**截距**，即 x=0 时的 y 值，整条线上下平移

### 2. 训练集的表示
- `x_train`：输入特征（如房屋面积，单位 1000 sqft）
- `y_train`：目标值（如房价，单位 1000 美元）
- `m`：训练样本数量，可用 `x_train.shape[0]` 或 `len(x_train)` 获取
- `x^(i), y^(i)`：第 i 个训练样本（Python 中从 0 开始索引）

### 3. Matplotlib 绑图
- `plt.scatter()`：画**散点图**，适合展示**离散**的训练数据点
- `plt.plot()`：画**折线/曲线**，适合展示模型的预测线
- 风格参数：`marker='x'` 设置标记形状，`c='r'` 设置颜色

### 4. NumPy 数组
- `np.array([1, 2])` → 一维数组 (2,)
- `np.array([[1, 2], [3, 4]])` → 二维数组 (2×2)，注意需要**双层方括号**
- `np.zeros(m)` → 创建 m 个元素的全零数组

## 出错的地方

### 单位换算错误
- **问题**：预测 1500 sqft 房价时，答成了 $310k
- **原因**：x 的单位是 1000 sqft，所以 1500 sqft → x = **1.5**（不是 1.05）
- **正确计算**：200 × 1.5 + 100 = **400**（即 $400k）
- **教训**：做预测时一定要注意数据的单位，先转换再代入公式

## 学生提问与解答

### Q1: `scatter` 是控制绘画风格的吗？
**A**: 不是。`plt.scatter()` 是一种**绑图函数**，专门画散点图。控制风格的是它的参数（如 `marker`、`c`）。与 `plt.plot()`（画线）是两种不同的绑图方式。

### Q2: 如何用 `np.array` 生成 2×2 矩阵？
**A**: 需要用嵌套列表（双层方括号）：
```python
# 错误 ❌
np.array([1,2], [3,4])

# 正确 ✅
np.array([[1,2], [3,4]])
```
外层 `[]` 表示整个矩阵，内层每个 `[]` 代表矩阵的一行。

## 关键代码

```python
def compute_model_output(x, w, b):
    m = x.shape[0]
    f_wb = np.zeros(m)
    for i in range(m):
        f_wb[i] = w * x[i] + b
    return f_wb
```
