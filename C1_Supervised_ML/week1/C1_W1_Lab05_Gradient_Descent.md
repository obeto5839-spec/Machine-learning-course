# C1_W1_Lab05 - 梯度下降 (Gradient Descent)

## 核心知识点

### 1. 梯度下降是什么
- 一种用于**最小化代价函数 J(w, b)** 的优化算法
- 通过不断更新参数 w 和 b，让 J 逐步减小，最终找到最优解

### 2. 更新公式
```
w = w - α * (∂J/∂w)
b = b - α * (∂J/∂b)
```
- **α（alpha）**：学习率，控制每一步的步长大小
- **∂J/∂w**：代价函数对 w 的偏导数（斜率），表示 J 在 w 方向上的变化率
- **∂J/∂b**：代价函数对 b 的偏导数

### 3. 偏导数的直觉理解
- 偏导数 = "固定其他变量不动，只看 J 随某一个变量变化的斜率"
- ∂J/∂w > 0 → 斜率向上 → w 需要减小
- ∂J/∂w < 0 → 斜率向下 → w 需要增大
- 梯度下降自动做到这一点：减去一个负数 = 加上正数

### 4. 学习率 α 的影响
- α 太小：收敛速度很慢
- α 太大：可能跳过最优解，甚至发散
- α 始终是正数

### 5. 同步更新
- w 和 b 必须**同时更新**（先算出两个新值，再一起赋值）

### 6. 线性回归的梯度下降公式
```
w = w - α * (1/m) * Σ(f(x⁽ⁱ⁾) - y⁽ⁱ⁾) * x⁽ⁱ⁾
b = b - α * (1/m) * Σ(f(x⁽ⁱ⁾) - y⁽ⁱ⁾)
```
- 注意：w 的更新公式末尾多乘了一个 x⁽ⁱ⁾，b 的没有

## 学生提问记录

- **Q**: 公式中 ∂J/∂w 的符号 ∂ 是什么？
- **A**: 是偏导数符号（partial derivative），读作"偏 J 偏 w"。当函数有多个变量时，用 ∂ 代替 d，表示只对其中一个变量求导，其他变量视为常数。
